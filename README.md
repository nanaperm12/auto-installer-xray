## Requirement OS :

- Ubuntu 18.04 / 20.04 x64 bit
- CPU MIN 1 CORE
- RAM 1GB

- VMESS WS TLS 443
- VMESS WS NON TLS 80
- VMESS GRPC 443
- VLESS WS TLS 443
- VLESS WS NON TLS 80
- VLESS GRPC 443
- TROJAN/TROJAN GO WS TLS 443
- TROJAN/TROJAN GO GRPC 443

- SHADOWSOCKS GRPC 443
